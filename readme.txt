Adds support to limit the number of Contact Form 7 form submissions per ip per period

== Description ==
Prevent bots from spamming your inbox or otherwise inconvenient behaviour by limiting the number of times a form can be submitted from a single IP address.
Simply configure the maximum number of submissions per period and the user will not be able to submit your Contact Form 7 forms anymore.

== Installation ==

* Go to your Dashboard
* Opt-in to Plugins > Add New
* Search for \"Rate limiting for Contact Form 7\"
* Click install and Activate

== Screenshots ==

1. Admin settings screen can be found under the Contact main menu item.

== Frequently Asked Questions ==

= Why would I need this plugin? =

Websites try and prevent bots from spamming form with all sorts of meassures; captcha's don't always block these bots and when they come through, your mailbox may get spammed with all sorts or unwanted messages. This plugin will limit the number of messages that come through.

= Does this plugin work with other form plugins? =

No, we have only built a version that integrates with Contact Form 7, if you need one for another plugin, feel free to contact us, maybe we can work something out.
